### Project 2: DJBHashFunction
<p> This project was my second time coding a hash function. I hadn't quite known about how hashing worked or why it was done. Yet, this project was really awesome. I enjoyed handling the large .txt file of common names and ordering them in 53 slots, as directed. I learned that the Daniel J. Bernstein Hash function is a way to store primitive data types as efficiently as possible in a customizable number of slots. When I say efficient, I mean that the simple DJB hash function allows hashed data to be accessed faster than most other hashing functions. My program also used a vector array instead of a standard array, since there wasn't a fixed size for how my data would be stored. Using the vector instead of the fixed array was intriguing to me because it allowed me to experiment with different applications between vectors and arrays.</p><br><br>
